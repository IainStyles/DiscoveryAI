# Discovery-AI

The course will cover a series of topics e.g.,

1.	Signature Discovery
    * Introduction: Data signatures are distinctive, domain-specific patterns that characterise objects, events, or processes.
    * Classical Methods: Techniques like clustering, feature selection, and discriminative modelling to identify salient patterns.
    * Modern Methods: Detection and representation learning approaches that characterise objects without relying solely on class labels.
    * Case Studies: Viral signature discovery in biomedical data for diagnosis and monitoring.

2.	Causality Discovery
    * Introduction: Distinguishing causation from correlation is essential for reliable inference and decision-making.
    * Classical Methods: SCMs, Granger causality, constraint-based (e.g., PC) and score-based (e.g., GES) approaches.
    * Modern Methods: Deep learning models like CausalFormer and world models for learning causal structure, interventions, and counterfactuals.
    * Case Studies: Drug target discovery in biomedical research.

3.	Hypothesis Discovery
    * Introduction: Hypothesis generation is central to exploration, experimentation, and theory building.
    * Classical Methods: Abductive reasoning to infer the most plausible explanations under uncertainty.
    * Modern Methods: LLMs and world models to suggest relationships, mechanisms, and counterfactuals.
    * Case Studies: Investigating biological mechanisms of ageing using AI-driven hypothesis discovery.
